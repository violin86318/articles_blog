ListenHub ASR 龙虾专用语音识别 API 全新上线，无限免费。

API 特点：

- 本地离线转录，无需 API Key，安装即可使用。

- 专为 Agent 设计，方便你的 Claude Code 和龙虾🦞直接接入自动化工作流。

- 我们采用双模型方案：默认 sensevoice 支持中英日韩粤等多语言识别，也提供轻量的 whisper-tiny.en 适配纯英文场景。

- 配合 ffmpeg 还能支持更多音频格式。

- 转录之后，还能无缝衔接 Podcast、Explainer 等内容工作流，把音频真正变成可复用内容。

免费安装，免费使用，让你的 Agent 耳聪目明！

安装地址： https://listenhub.ai/docs/zh

此外 ListenHub 的 API 现已支持六种能力，特别为 Agent 优化，六种能力分别是：

- 播客

- TTS 文本转语音

- 解说视频

- AI 图片生成

- ASR 语音识别

- 内容解析

ListenHub 会员可直接接入你的 Agent 进行使用，随意畅玩。
