正文整理

开源一个我写小说的skill，顺便说下我的搭建思路和使用方法

不是一个skill，应该说是一组skill。

我已经用这一组skill写了两篇长篇小说和三部短篇小说。平台审核是肯定能过的，关键是你自己读完就能感受到，确实已经算不错的小说了。

只是我自己感觉写小说有点浪费时间，坚持不住，所以就不写了。本着开源的精神，今天把这个分享给大家。

建设背景

本人喜欢写作，但是文笔不是很好。脑子里面天花乱坠，但是下笔空空如也，半天憋不住一个字来。

感谢AI的到来，让我能够圆了我的写作梦。我开始着手炼化我的skill。

建设思路

小说skill组一共五个，分别是小说大纲调研工具

（novel-outline-researcher）、小说写作工具（

novel-studio）、小说问题检查工具（novel-qa）、小说风格制作工具（novel-style-reference）和个人风格工具（yunhui-style-writer）。

小说大纲调研工具（novel-outline-researcher）主要是帮助我开题，不断提问问题完善我的思路。也可以卡文或者要对大纲重构的时候用。

小说写作工具（novel-studio）是核心工具，它可以按照我的思路写大纲、写人物小传、写时间线、甚至规定空间布置。然后再根据这些内容开始完成每一章节的小说，而且写小说过程我也给他定了很多规矩，并且要求自我学习。可以说，这个skill在我的不断优化下基本可以独立使用了。

小说问题检查工具（novel-qa）是专门用来检查小说前后矛盾的地方、场景描述是否一致、时间线是否有问题、人设是否一致、AI味等内容。并生成相应的检查报告。

小说风格制作工具（novel-style-reference）是通过给他读一些小说，让他总结这个作者的小说风格，然后在你写小说的时候可以让他按照这种风格进行写作。我总结了村上春树和川端康成的写作风格，还总结了王家卫的写作风格。也可以按照小说来进行总结。

)

个人风格（yunhui-style-writer）也就是自己的风格，就是按照自己的写法总结的。

以上五个skill分工合作，加上我自己的总体把控，就能产出一部一部完整的小说来。

再多说几句这个核心工具novel-studio，我给他在文风自然度上面设计了很多内容，也制定了铁律。但是有些大纲内容它确实还是很难驾驭，写着写着还是会飘，所以小伙伴们如果实在调整不过来，那别试了，直接手写吧。

)

使用方法

写小说分三步走，第一步：构思小说，第二步：制定大纲，第三步，写各章节。

构思小说是在你脑子里面完成的，你思考你大概要写什么类型、什么题材、长篇还是短篇，男女主人设等等。

有个雏形后就开始用novel-outline-researcher来给你完善你的想法，然后形成报告。

这一步完成后你再用novel-studio，将上面的报告给到这个skill，它就会按照报告开始创建小说的介绍（meta）、大纲（outline）、时间线（timeline）、人设（characters）以及重要场景（spaces）。

)

其中大纲不是一蹴而就的，它会先分有几幕，每一幕讲什么。然后先出前五章的大纲。这个时候你通过反复检查和确认，用自然语言让AI按照你的意思调整，最终直到你满意为止。

PS：如果大纲一直调整不满意，你可以直接写，写进去复制进去就行。我到了小说后期都是手写大纲，然后让AI去生成文章的。不然AI写的大纲确实不咋地，越往后越不行。

最后一步就是开始写小说了。在大纲里面可以要求每一章字数，这个看个人喜好。如果大纲内容撑不起你的字数，那可能就要凑字数了。。。这是本人亲历的教训。

写完后你要读一遍，自己出的小说自己都懒得读，就不要指望别人读了。而且不止读一遍，要反复和AI商量怎么改，改到满意。这才是你产出的作品，而不是AI出的了。

novel-studio这个skill会结合上下文学习你不满意的地方，那些字不要用，然后去写下一章内容。

)

novel-studio写完也会自检，你也可以调用novel-qa这个检查。我一般用的少，因为我自己读过了，如果你不读，都是AI完成，那这几个skill可以配合完成。

最后

以上就是我这一组小说写作skill的建设思路和使用方法。skill我已经上传到Github，地址是：https://github.com/zhengyunhui123-dev/cursor-claude.git  也可以后台回复“小说”直接获取。

然后我还想说点的是“尽信书不如无书”，大家使用的时候可以根据自己的喜好修改skill，让其更好地为您服务。

另外还有一个误区是不要觉得用这个可以自动化产出一部完整的小说，每个环节还是需要人工干预的。

它只是节约了你码字的时间，让你把更多的时间投入到思考上。也给了一些像我这样有“写小说梦”但是文笔很差的人一个机会，仅此而已。

最后，祝愿大家都能写出爆款小说来。
