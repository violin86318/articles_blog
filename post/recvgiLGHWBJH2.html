39万播放的AI仿真人短剧，最值钱的是这套可复用SOP

这两天，我看很多人还在群里聊 Seedance 2.0 排队、即梦涨价、模型报错。

这些当然都重要。

但我们团队把一集 AI 仿真人短剧从头到尾又跑了一遍之后，一个感受越来越强。

真正拉开差距的时间点，往往发生在生成视频前。

同样是女频爽剧题材，有人拿着模型反复抽卡，有人已经把同题材跑到单集 39 万+ 播放，顺手把后面的复用流程也沉淀下来了。

我们也会在明天晚上八点，给大家直播点评这一期 AI 仿真人短剧训练营作品，有不少家人们做得很惊艳，欢迎预约。

差距通常不在那 15 秒视频里，差距更早。

剧本有没有拆成镜头语言，角色资产有没有按状态分开建，场景和道具有没有提前备够，人物站位有没有先替模型想清楚。

这些东西如果前期没想好，后面做得越多，返工往往越多。

所以这篇我不想讲很空洞的概念，我直接把我们这套真实案例里，最值钱的 SOP 主干拆给你看。

若需要完整的制作 sop 内容，可以扫文末二维码了解，单篇 299，或者加入下一期训练营可以免费获得。

这个案例本身也很有代表性。

题材是典型的古风女频爽剧，里面有重生、宅斗、打脸、复仇，情绪浓度很高，前几天在平台上已经跑出了 39 万+ 播放。

这种题材很适合 AI 仿真人短剧，也特别考验制作流程。

先看三组最容易把人拉开差距的对比

如果你现在正在做 AI 仿真人短剧，我很建议你先看完下面这三张图。

很多人为什么后面越做越乱，答案基本都藏在这里。

1. 同一套角色提示词，不同模型跑出来差很多

我们这次把即梦几个常用版本都跑了几遍。

同一个角色、同一套提示词，最后出来的人物气质、服装克制感、脸部稳定性，差异其实非常大。

我们最后的感受也很明确，4.6 这一版在古装服化道的克制感和稳定性上更好，所以后面很多角色都沿着这个方向继续打磨。

)

2. 同一个角色，也要按剧情状态拆成不同资产

沈宁这个角色，我们至少拆成了两套。

一套是重生前病弱、落魄、钗环尽卸的状态。

一套是重生后重新掌住局面、眼神和气场都稳下来的状态。

如果把这两个状态混成一套，视频里的人物会非常容易不稳定。

)

3. 场景也不能随便出一张“差不多”就开拍

这次柴房场景，我们前面至少筛了几轮。

有的版本太破，有的版本空间关系不对，有的看起来甚至像乡下杂物间，根本撑不起后面的对话和人物站位。

直到场景能同时满足题材气质、正反打需求和空间关系，我们才敢继续往下走。

)

我们这次复盘后，重新把整套链路压缩成了一张图。

你可以先把它记住。

后面真正做视频的时候，基本就是沿着这条链往下走。

)

第一步，先把剧本拆成能执行的镜头节奏

普通小说是写给人脑补的，短剧分镜是写给镜头执行的，这两种语言差得很远。

所以我们这次一上来根本没有先出图，第一步先拆剧本。

这集内容最后被我们压成了三段。

第一段是前世惨死和重生觉醒，开头就把雷雨夜、柴房、灌毒酒这些最狠的情绪甩出来，先抓观众。

第二段是反派上桌施压，陆老夫人和陆远铮来骗对牌和嫁妆钥匙，沈宁不急着翻脸，先演给他们看。

第三段才是抢钥匙、揭露外室、债主上门，把这一集的爽点和下一集的钩子一起立住。

你会发现，剧本一旦这样拆清楚，后面很多决定都变容易了。

角色为什么要分状态建立，场景为什么至少准备两个视角，长台词为什么必须拆段，全部都有了依据。

第二步，角色资产别只停在一张脸

很多人做 AI 仿真人短剧，最容易偷懒的地方就是角色，随便跑几张图，看起来差不多，就直接进视频。这一步最容易把后面全带偏。

这次案例里，AI 最开始提出来的主要角色有沈宁、陆远铮、陆老夫人、林娇娇和丫鬟翠儿。

但这还远远不够，真正能进视频的角色资产，至少得回答一件事。

这个角色在不同剧情阶段，到底长什么样。

比如沈宁，就至少要拆成两套。重生前，她是病弱、落魄、钗环尽卸的状态。

)

重生后，她是能压住全场、眼神和气场都更端庄稳重的状态。

)

如果把这两个阶段混成一套资产，视频很快就会混乱。

而且古装题材特别考验审美，AI 很容易把人做得过于塑料，过于丑，不符合实际历史。

)

我们这次后面是直接去找对标审美，把服化道往更克制的宋制路线去调，衣服颜色、头饰、发型、妆感，都是一点点打磨出来的。

)

女配林娇娇也是一样，不能只让 AI 按刻板印象往妖艳上跑。

她最后的角色资产图，是在多轮调整服装、发型、脸型之后才慢慢出来的。

)

)

角色资产做好之后，还有一个隐藏价值。

你后面不管换模型、换镜头、补镜头，都会轻松很多。

因为你终于拿到的是一套可以复用的资产，不是一次性的好运气。

第三步，场景、道具和站位是生成阶段的基础设施

角色定下来之后，接着就是场景和道具。

这次剧本的核心场景其实就两个，一个是侯府柴房，一个是重生后的侯府正院会客厅。

)

很多人做到这里会觉得，一张差不多的古代大厅图也够用了。

真做短剧，通常不够。

因为你后面还要做正反打、人物对峙、视角切换、过肩镜头。

所以我们这次没有只做一张场景图，而是把关键视角都提前准备出来。

柴房正面、望向门口的反打、侧面全景，正厅中轴视角、正门望院视角，这些都提前准备好。

)

)

这样模型在生成时，才有足够稳定的空间信息可以参考。

更关键的是人物站位。

很多人只盯着脸像不像，结果真正翻车的往往是空间关系。

为什么别人做出来的对话戏很自然，你做出来的人一会站左边，一会又飘到右边。

因为视频模型不会自动替你做导演。

室内戏、多人物同框、正反打场景里，提前做一张人物位置关系图，价值非常大。

)

你把它喂进去，模型对空间关系的理解会稳很多。

第四步，长台词拆段，废镜头抢救，才是交付差距真正开始拉大的地方

很多人前面都还能勉强跟上。

真正开始大量翻车，通常是从这里开始。

一大段台词直接往 15 秒素材里塞，语速一下就像在念 rap，吞词、改词、嘴瓢，全来了。

这次案例里，陆老夫人、陆远铮和沈宁有一段连续拉扯的戏，正常语速读下来至少二十秒以上。

)

我们后面直接把它拆成了两段。

第一段是陆老夫人施压，沈宁表面乖巧地听。

第二段是陆远铮附和，沈宁内心腹诽。

)

这么一拆，素材稳定性一下就上来了。

再往后，就是废镜头抢救。

这一步特别重要。

很多看起来已经废掉的素材，其实不一定真要扔。

有些镜头动作和构图都不错，只是背景里多了人，或者道具不对，或者远景脸崩了。

这类镜头完全可以先保住首帧或尾帧，再重新图生视频，或者局部修补之后接回去。

)

我们这次也把可灵、Vidu、Seedance 放在不同问题里做了对比。

可灵清晰度高，Vidu 短表演更灵，Seedance 综合能力最全，但很吃前面的资产准备。

真正做项目时，核心不是押注某一个模型。

核心是你得知道，哪类问题该用哪种方式补。

为什么这套流程能一直复用

过去两年，我们在做 AI 视频商单、短剧项目和内容号的过程中，越来越在意一件事。

一次偶然出片，说明不了太多，下次还能不能稳稳做出来，才真正值钱。

这套 SOP 的价值也在这里，它把随机性往前减少了。

剧本先拆好，角色先立住，场景和站位先准备，长台词先分段，坏镜头有补救方案，后期知道怎么把情绪收住。

这样你拿到的就不只是一条视频。

你拿到的是一套可以持续复制、工业生产的交付方式。
