设计与视觉 PPT HTML Skill 约 2 分钟阅读

每天一个好用skill分享-用html做精美ppt

一句话总结

Frontend Slides能生成零依赖的HTML演示文稿，通过独创的“Show, Don't Tell”流程预览风格。

核心摘要

Frontend Slides 是一款专为Coding Agent设计的开源工具，通过一句话指令即可生成零依赖的HTML幻灯片（纯内联HTML/CSS/JS）。其核心优势包括： 1. 无需环境：直接浏览器运行，摆脱npm/构建工具依赖； 2. 智能设计：内置12+精选风格预设（含暗色/亮色/特殊主题），独创“Show, Don't Tell”流程——自动生成3套预览供用户选择，避免抽象描述； 3. 极致适配：采用clamp()响应式缩放，严格适配100vh视口，确保全屏幕无溢出； 4. 灵活模式：支持从零新建、PPTX一键转HTML、增强已有演示； 5. 高效编辑：自动拆分内容密度，浏览器内直接修改文字。 提供Claude/Cursor/OpenClaw一键安装，触发指令：“帮我做一份XX演示文稿”。

正文整理

文章标题为《每天一个好用skill分享-用html做精美ppt》，内容主要介绍了一种名为Frontend Slides的开源Skill，该Skill专为Coding Agent设计，能够生成零依赖的HTML演示文稿。

文章指出，传统的演示文稿制作方式往往使用固定的PPT模板，或者使用Reveal.js/Slidev等工具，但这些都需要安装许多依赖。而Frontend Slides则不同，它能够通过一句话生成纯HTML、CSS和JS内联的演示文稿，无需npm或构建工具，直接在浏览器中打开即可使用。

文章还介绍了Frontend Slides的核心亮点，包括内置的12+精选风格预设，涵盖暗色、亮色和特殊主题等，能够避免AI生成的千篇一律的设计。此外，该Skill还独创了“Show, Don't Tell”流程，即先自动生成3套风格预览，用户看了效果后再选择，无需描述抽象的偏好。每张幻灯片都严格适配100vh视口，使用clamp()进行响应式缩放，确保在任何屏幕上都能完美呈现，不会溢出。

Frontend Slides支持三种模式：从零新建、PPTX一键转HTML、增强已有演示。它还提供了6种幻灯片类型，包括标题、内容、网格、代码、引用和图片，能够满足不同的演示需求。此外，该Skill还能自动控制内容密度，超出限制时会自动拆页，并支持在浏览器内直接编辑文字。

安装方面，Frontend Slides提供了Claude Code一键安装、OpenClaw命令安装和Cursor拉取即用的方式，用户只需说一句“帮我做一份XX演示文稿”即可触发。

文章最后提供了该Skill的详细信息和更多基础知识的学习链接，方便用户了解更多。
