不写代码如何榨干模型 token plan 价值？

大家好，我是空格，换个新头像冒个泡，顺便分享些有用的

身边订阅模型 API 的人越来越多了，但发现很多人一个月用不到 1/10 额度。

怎么让 Agent 24 小时来产出，人休息的时候它还在工作？

我的方法是用 Skill 来扩应用场景：

分为主动（定时跑任务）和被动的任务（有需求再找他）

主动型：
1 每天早上 RSS 总结。
我用的是 rss-aggregator 这个 Skill，把订阅的博客和资讯源博主地址喂给它。早上一睁眼信息就在手机里了。
github.com/zephyrwang6/info-skill

2 播客自动处理。
关注的 YouTube 频道一更新，先用扫最近 48 小时的新视频，再把字幕拉下来翻成中文，最后跑一遍摘要、金句提取，全部塞进飞书知识库。我醒来直接看精华。

github.com/zephyrwang6/info-skill

3 选题素材采集。
每天扫一遍 X、即刻、小红书的关键词，把跟我领域相关的爆款内容抓回来归档，顺手生成选题候选。
早上醒来，有值得研究的就深入看

github.com/zephyrwang6/info-skill/tree/main/topic-collector

被动型：
尽可能找稳定的skill 场景。
一种是我自己做，比如PPT、原型图、流程图、架构图，这些我都有分享过

还有一些是第三方软件提供的 Skill，可以直接让你的 Agent 来操作 SaaS 工具

1 飞书 lark cli
飞书官方出品，200 多个命令加 23 个 AI Agent Skill。日历、文档、电子表格、多维表格、IM、知识库、视频会议任务全覆盖。
github.com/larksuite/cli

2 钉钉 dingtalk-workspace-cli
钉钉官方 CLI，文档、AI 多维表格、群聊、会议室、日程、待办全在里面。
github.com/DingTalk-Real-AI/dingtalk-workspace-cli

3 MiniMax 官方 skill
16 个Skill。 多模态 生成、 语音、音乐、视频、图片、GIF 表情包、PPTX、DOCX、XLSX、PDF。前端、安卓、iOS、Flutter、React Native 也全。
github.com/MiniMax-AI/skills

4 Agents365-ai/drawio-skill
用 Agent 出drawio 文件，如 ERD、UML、时序图、架构图、ML 网络图、流程图六种。
github.com/Agents365-ai/drawio-skill

5 Agents365-ai/video-podcast-maker
B 站和 YouTube 视频播客一站式生成
github.com/Agents365-ai/video-podcast-maker

6 科大讯飞 SkillHub
企业级 Skill 。RBAC、审计、版本管理，Docker 一键部署
github.com/iflytek/skillhub

7 wpsnote-skills
WPS 的 40 多个 Skill
github.com/wpsnote/wpsnote-skills
