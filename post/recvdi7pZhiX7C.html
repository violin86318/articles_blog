告别飞书和QQ！OpenClaw接入微信才是天选组合！（保姆级教程）

兄弟们！

你们安装小龙虾（OpenClaw）之后，在接入聊天软件时是不是都会纠结？

现在OpenClaw支持：飞书、QQ、企业微信、钉钉、Telegram、Discord。

企业微信和钉钉是办公软件，后两个一般用户使用不了。

相比飞书和QQ，微信才是大家日常用的最多的聊天工具。

刚好想起前天看到江枫老师写的微信接入OpenClaw的文章，试着操作了下。

调通了，OpenClaw+微信才是天选组合。

可以在微信中直接和OpenClaw助理对话，我给他设定了个定时任务，每天帮我推送深圳的天气。

这个方案的原理是使用企业微信的微信插件能力，让用户可以在手机微信APP上和企业微信AI主理对话聊天。

二、OpenClaw配置成公网

首先说下，我是使用腾讯云服务器来配备的（阿里云也是支持的），本地部署方法据说会比较麻烦（设置NAT穿越）。

本身云服务就有公网IP，我们可以把服务器配置成公网地址。

图片：来源内容未完整抓取

打开OpenClaw.json文件，修改下面几个配置：

图片：来源内容未完整抓取
  1. bind-->修改成lan
  2. allowedOriginsd-->修改成你的公网IP
  3. allowInsecureAuth-->修改 true,
  4. dangerouslyDisableDeviceAuth-->修改 true

修改参数后，用HTTP的方式也能访问，但是出于安全性不要泄露自己的IP以及token。

修改好之后，运行重起指令让配置生效。

openclaw gateway restart

进入WEBUI页面，在快照地方看到状态是绿色的正常，说明配置成功了。

图片：来源内容未完整抓取

三、安装微信插件

安装openclaw官方微信插件，运行下面的指令：

openclaw plugins install @openclaw-china/wecom-app

等待一会时间，插件安装完成。

2、为OpenClaw配置通道

在腾讯云应用管理控制台，进入应用详情页，在OpenClaw配置面板中，添加企业微信应用通道，需要配置五个参数。

Corp ID

在企业微信管理后台中，找到“我的企业-企业信息”页面，在页面底部找到企业ID并且复制，后续步骤中需要使用。

图片：来源内容未完整抓取

将复制好的企业ID，粘贴至OpenClaw配置面板中的对应位置（Corp ID）输入框。

图片：来源内容未完整抓取

Corp Secret

在企业微信管理后台中，找到刚才创建好的OpenClaw应用，进入应用详情。

图片：来源内容未完整抓取

在应用详情页内，找到Secret，点击查看按钮，在弹窗中点击发送。

图片：来源内容未完整抓取

企业微信会收到一条消息提醒，在这个界面内复制Secret。

图片：来源内容未完整抓取

将复制好的Secret，粘贴到OpenClaw配置面板中的**【Corp Secret】**输入框中。

图片：来源内容未完整抓取

Agent ID

在企业微信应该管理中，找到Agent ID，复制并粘贴到OpenClaw配置面板中的**【Agent ID】**输入框中。

Token

在企业微信应用管理详情页【接收消息】，点击【设置API接收】。

图片：来源内容未完整抓取

点击随机获取Token和EncodingAESKey，然后将这两个参数分别填入OpenClaw配置面板的对应位置输入框中。

这五个参数全部填写完成后，单击下方的添加并应用按钮并确认操作。
