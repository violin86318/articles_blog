📌 Medeo 接入 OpenClaw 官方实操教程

全程就3步，10分钟就能搞定，每一步都给大家标清楚了，还配了截图指引，跟着做，不用担心踩坑！

PS：操作前提是飞书已经安装了 OpenClaw 生态

关注 Medeo 视频号了解更多使用教程

第一步｜安装 Medeo 技能并获取 API 密钥

  • 将以下内容输入给 OpenClaw 机器人，它会进行自动配置安装。

请安装 medeo-video 技能，GitHub 地址：https://github.com/one2x-ai/medeo-video-skill

安装完成后，你的 OpenClaw 机器人将引导你注册 Medeo 平台并获取 API 密钥（以 mk_开头）。配置完成后，只需说出 “帮我生成一个 XX 视频” 即可使用。

相关链接：

  • GitHub地址：https://github.com/one2x-ai/medeo-video-skill

  • Medeo 平台：https://medeo.app

  • OpenClaw 注册：https://openclaw.ai

)

  • 安装完成后，它会提醒你获取 API Key。点击链接 https://medeo.app/dev/apikey ，即进入获取 API Key 界面。

)

第二步｜在 Medeo 平台创建并获取 API 密钥

  • 点击第一步中的 apikey 链接，进入到 API Key 界面。接下来，点击「Create API Key」按钮，启动 API 密钥创建流程。

)

  • 输入自定义密钥名称后，点击「Create」完成创建。

)

  • 生成 API 密钥后，点击复制按钮保存。

)

第三步：测试使用（快速上手，2分钟搞定）

  • 将复制保存的 API Key 发送给 OpenClaw 机器人，它会帮你自动配置。

)

  • 配置成功后，我们只需要和 OpenClaw 机器人进行对话就能开始创作，无需切换工具、学习复杂操作，用大白话对话就能调用 Medeo 生成视频，全流程在办公场景内一站式完成！

)
