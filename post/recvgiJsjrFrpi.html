ListenHub CLI 发布并开源，AI 音乐首次支持

一句话总结

ListenHub CLI 开源发布并首次支持 AI 音乐生成功能。

核心摘要

ListenHub CLI 正式开源发布，支持通过 npm install -g @marswave/listenhub-cli 全局安装。本次升级新增两大AI功能： 1. AI音乐生成：输入文字描述即可创作音乐（例：listenhub music generate --prompt "动感电子舞曲"），支持 --instrumental 纯音乐参数。翻唱功能可处理本地音频（mp3/wav/flac/m4a/ogg/aac，≤20MB）。 2. PPT幻灯片制作：根据主题自动生成带配音幻灯片（例：listenhub slides create --query "TypeScript入门指南"），可定制模板/语言/尺寸/页数。 原有功能保留： 播客（quick/deep/debate模式） TTS文本转语音（--lang "zh"支持多语言） 解说视频 图片生成（支持多张本地参考图） 技术特性： 所有命令支持 --json 输出和 --no-wait 异步模式 底层开源 TypeScript SDK（@marswave/listenhub-sdk），含OAuth登录/自动刷新token/429重试 MIT协议开源，安装后需 listenhub auth login 授权登录 GitHub链接详见原文。

正文整理

文章标题为“ListenHub CLI 发布并开源，AI 音乐首次支持”。

文章内容主要介绍了 ListenHub CLI 的发布和开源情况。该工具可以通过 npm install -g 安装，并在各种 Agent 环境中直接运行。除了之前已有的播客、TTS（文本转语音）、解说视频功能外，这次还新增了 AI 音乐生成和 PPT 幻灯片制作能力。

在 AI 音乐生成方面，用户可以通过输入一段文字描述来生成相应的音乐作品。例如，使用命令 listenhub music generate --prompt "动感电子舞曲" --style "EDM" --title "夜间飞驰" 可以生成一首具有动感电子舞曲风格的曲子。如果需要纯音乐，可以添加 --instrumental 参数。此外，还提供了 cover 翻唱功能，用户可以上传本地音频文件进行翻唱，支持的音频格式包括 mp3、wav、flac、m4a、ogg、aac，最大支持 20MB 的文件大小。

在 PPT 幻灯片制作方面，用户只需提供主题，即可生成一套带配音的幻灯片。例如，使用命令 listenhub slides create --query "TypeScript 入门指南" 可以生成关于 TypeScript 入门指南的幻灯片。SDK 中还提供了更多细节，可以指定模板类型、语言、尺寸、页数等参数。

除了新增功能外，播客、TTS、解说视频等原有功能也都在。播客支持 quick、deep、debate 三种模式，用户可以输入参考链接来进行创作。TTS 功能可以通过一行命令完成，例如 listenhub tts create --text "你好世界" --lang "zh"。

此外，图片生成功能也支持传入本地参考图，最多可以传入多张图片。例如，使用命令 listenhub image create --prompt "水彩风格的小龙" --reference ./sketch.jpg --reference ./palette.png --aspect-ratio 16:9 --size 4K 可以生成一张水彩风格的小龙图片。

所有创作命令都支持 --json 输出和 --no-wait 异步模式，方便用户编写脚本进行串联。SDK 底层也一起开源了，使用 TypeScript 编写，包名为 @marswave/listenhub-sdk。SDK 提供了 OAuth 登录、自动刷 token、429 重试等功能，方便用户在自己的项目中集成 ListenHub 功能。

安装和更新 ListenHub CLI 的命令为 npm install -g @marswave/listenhub-cli。安装完成后，需要先登录，可以使用命令 listenhub auth login 进行登录，浏览器会弹出授权页面，授权后 token 会存储在 ~/.config/listenhub/credentials.json 文件中，过期会自动刷新。更新时，只需重新安装最新版本的 CLI 即可。

CLI 和 SDK 都遵循 MIT 协议，用户可以自由使用。

文章最后提供了 CLI 和 SDK 的 GitHub 链接，方便用户获取更多信息。
