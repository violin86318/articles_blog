ListenHub CLI 发布并开源，AI 音乐首次支持

❝ ListenHub CLI 开源发布并首次支持 AI 音乐生成功能。

以下是文章核心内容提炼（196字）：

ListenHub CLI 正式开源发布，支持通过 npm install -g @marswave/listenhub-cli 全局安装。本次升级新增两大AI功能：
1. AI音乐生成：输入文字描述即可创作音乐（例：listenhub music generate --prompt "动感电子舞曲"），支持 --instrumental 纯音乐参数。翻唱功能可处理本地音频（mp3/wav/flac/m4a/ogg/aac，≤20MB）。
2. PPT幻灯片制作：根据主题自动生成带配音幻灯片（例：listenhub slides create --query "TypeScript入门指南"），可定制模板/语言/尺寸/页数。

原有功能保留：
- 播客（quick/deep/debate模式）
- TTS文本转语音（--lang "zh"支持多语言）
- 解说视频
- 图片生成（支持多张本地参考图）

技术特性：
- 所有命令支持 --json 输出和 --no-wait 异步模式
- 底层开源 TypeScript SDK（@marswave/listenhub-sdk），含OAuth登录/自动刷新token/429重试
- MIT协议开源，安装后需 listenhub auth login 授权登录
