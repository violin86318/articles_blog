ListenHub CLI 发布并开源，AI 音乐首次支持

ListenHub CLI 发布并开源了。npm install -g 装完就能用，各种 Agent 里直接跑。

这次除了之前就有的播客、TTS、解说视频，新加了两个能力：

AI 音乐生成和 PPT 幻灯片。

先说音乐。

音乐生成

给一段文字描述，直接出一首曲子：

listenhub music generate \
  --prompt"动感电子舞曲" \
  --style"EDM" \
  --title"夜间飞驰"

想要纯音乐、没人声的，加个 --instrumental：

listenhub music generate --prompt"宁静的钢琴旋律"--instrumental

还有个 cover 翻唱功能，丢一段本地音频进去就行。支持 mp3、wav、flac、m4a、ogg、aac，最大 20MB。CLI 会自动上传到云端再调接口，不用你操心：

listenhub music cover --audio ./original.mp3 --title"我的混音"

URL 也行，直接透传：

listenhub music cover --audio https://example.com/song.mp3

PPT 幻灯片

给个主题就能生成一套幻灯片，带配音的那种：

listenhub slides create --query"TypeScript 入门指南"

SDK 里能看到更多细节，可以指定模板类型、语言、尺寸、页数这些：

const { episodeId } \=awaitclient.createSlides({
  query: 'Introduction to TypeScript for JavaScript developers',
  imageConfig: { size: '2K', aspectRatio: '16:9' },
  template: {
    type: 'storybook',
    mode: 'slides',
    speakers: [narrator.speakerInnerId],
    language: 'en',
    size: '2K',
    aspectRatio: '16:9',
    pageCount: 2,
  },
});

其他能力

播客、TTS、解说视频这些老能力都在。

播客支持 quick、deep、debate 三种模式，可以丢参考链接进去：

listenhub podcast create \
  --query"2026年AI趋势" \
  --mode deep \
  --source-url https://example.com/article \
  --lang zh

TTS 一行命令搞定：

listenhub tts create --text"你好世界"--lang zh

图片生成支持传本地参考图，最多可以传多张：

listenhub image create \
  --prompt "水彩风格的小龙" \
  --reference ./sketch.jpg \
  --reference ./palette.png \
  --aspect-ratio 16:9 --size 4K

所有创作命令都支持 --json 输出和 --no-wait 异步模式，方便写脚本串联：

ID=$(listenhub music generate --prompt "测试" --no-wait --json | jq -r '.taskId')
listenhub music get "$ID" --json

SDK

底层 SDK 也一起开源了，TypeScript 写的，包名 @marswave/listenhub-sdk。

OAuth 登录、自动刷 token、429 重试这些都处理好了。装上就能在自己的项目里集成：

npm i @marswave/listenhub-sdk

用起来很直白：

const client = new ListenHubClient({
  accessToken: 'your-token',
  maxRetries: 2,
});

// 生成音乐
const task = await client.createMusicGenerate({
  prompt: 'Upbeat lo-fi hip hop beat with jazzy piano chords',
  style: 'lo-fi',
  title: 'Late Night Study',
});

// 创建播客
const { episodeId } = await client.createPodcast({ ... });

// 生成图片
await client.createAIImage({ ... });

SDK 的 repo 里有完整的示例代码，播客、TTS、解说视频、幻灯片、音乐、图片每个都有。

安装和更新

Node.js >= 20，然后：

npm install -g @marswave/listenhub-cli

装完先登录：

listenhub auth login

会弹浏览器授权，token 存在 ~/.config/listenhub/credentials.json，过期自动刷新。

更新的话重新装一遍就行：

npm install -g @marswave/listenhub-cli@latest
