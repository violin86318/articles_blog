开源我的所有 skill，以及 skill 管理方法

假期也没有闲着，今天在家办公，发现现在离开了Skill，根本没有办法工作，为了让自己在家里电脑也能用工作电脑的 skill，我把 100 多个 skill 做了一个同步和分类。

可以实现跨工具、跨电脑、跨网络来使用和分享 skill。

主要用到的方法就是用.claude/skills 来做全局 skill，其他工具如 codex 的skill 根目录做软链接同步.claude/skill。

再对全局 skill 按类型拆分文件夹，并在 github 建仓同步，这就实现了跨网络/电脑/工具同步了，最后，再建立一个 skill-mannage 来定时自动推送 git。

对于 git 的处理，强烈建议 GitHub CLI，可以实现让 Agent 无碍障增删改查 git。

既然做了分类，就给大家分享一下我的四大类 skill，分类清晰更便于各位使用。

共 100 多个 Skill 分了写作类、画图类、信息获取类、产品经理类，建了四个仓库

1 写作类，write-skill，21 个 Skill，覆盖内容创作全流程：

地址：https://github.com/zephyrwang6/write-skill

2 画图类draw-skill，13 个 Skill，解决"AI 配图"全场景：

地址：https://github.com/zephyrwang6/draw-skill

3 信息获取类：info-skill，14 个 Skill，把散落各处的信息收回来。

地址：https://github.com/zephyrwang6/info-skill

4 产品经理类：protodesign，13 个 Skill，产品经理的工具箱：

地址：https://github.com/zephyrwang6/protodesign

所有 skill: https://github.com/zephyrwang6/myskill

对于skill工作 的创造和管理，我已经在社群里分享过很多，如果大家感兴趣，可以点赞支持一下，后面再发长文分享制作技巧和管理方法，让你的 Agent 更有用
