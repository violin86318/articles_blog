每天一个好用skill分享-用html做精美ppt

做演示文稿，PPT 模板翻来覆去就那几套，用 Reveal.js / Slidev 又要装一堆依赖；让 AI 直接生成，出来的幻灯片千篇一律毫无设计感。
Frontend Slides 就是来解决这个问题的。

它是一个专为 Coding Agent 设计的开源 Skill（GitHub 13K+ Stars），能一句话生成零依赖的 HTML 演示文稿——纯 HTML + CSS + JS 内联，无需 npm，无需构建工具，浏览器打开即用。

【核心亮点】：内置 12+ 精选风格预设，涵盖暗色（Bold Signal、Neon Cyber）、亮色（Notebook Tabs、Split Pastel）、特殊主题（Terminal Green、Swiss Modern）等，杜绝"AI Slop"审美。独创"Show, Don't Tell"流程——先自动生成 3 套风格预览，你看了效果再选，不用描述抽象偏好。每张幻灯片严格适配 100vh 视口，用 clamp() 做响应式缩放，任何屏幕完美呈现，永不溢出。

支持三种模式：从零新建、PPTX 一键转 HTML、增强已有演示。
6 种幻灯片类型（标题/内容/网格/代码/引用/图片）。
内容密度自动控制，超出限制自动拆页。
还支持浏览器内直接编辑文字。

Claude Code 一键安装、OpenClaw 命令安装、Cursor 拉取即用——说一句"帮我做一份 XX 演示文稿"即可触发。

【skill详情】：skillsbook.fun/skill-frontend-slides.html
