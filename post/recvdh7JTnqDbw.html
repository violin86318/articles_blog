Obsidian 推出了 Defuddle， 把 Obsidian Web Clipper 推到了一个新的高度

❝ Defuddle 作为免费开源的 Obsidian 工具，让用户完全掌控网页内容清洗和 Markdown 转换，强化了本地优先理念，并挑战付费服务如 Jina.ai。

以下是文章核心提炼（196字）：

Obsidian 坚守“本地优先”理念，以纯文本 Markdown 文件构建用户自主掌控的笔记生态。 其核心工具 Web Clipper 插件可将网页内容（含元信息）转为 Markdown 存入本地库。最新推出的 Defuddle.md 将该功能提升至新高度，成为生态内强力工具。

Defuddle 本质是 Web Clipper 的网页版，提供三种使用方式：
1. 普通用户：通过浏览器访问网站，输入网址即可获取清洗后的 HTML/Markdown。
2. 开发者/AI Agent：通过 URL 接口直接获取清洗结果。
3. Obsidian 用户：直接使用 Web Clipper 插件无缝集成。

关键优势：
* 免费开源（对比竞品如 Jina Reader API 的 0.15美元/次收费）。
* 支持本地部署，用户完全掌控数据与流程，践行“工具服务于人，而非奴役于人”的理念。
