AI短剧太火了！建议收藏这几个GitHub项目，做第一波吃螃蟹的人

一条AI短剧，41.8万赞。一个人，没团队，全靠AI出片。----纯干货建议先收藏

2026年的数据已经不讲道理了：《斩仙台AI真人版》六天破亿，流水过百万的AI短剧有10多部，整个漫剧市场奔着300亿去。制作费砍掉九成，有团队一个人就能搞定从画图到成片的全流程。

这种"一人剧组"的能力，有没有开源方案？还真有。GitHub上扒到三个项目，都能跑通剧本到成片的完整链路。

1. 魔因漫创 Moyin Creator——工业级流水线

GitHub： github.com/MemeCalculate/moyin-creator

这个项目走的是正经"AI制片厂"路线。

桌面端应用，深度绑定字节的Seedance 2.0模型，把短剧制作拆成了五个标准化环节：剧本→角色→场景→分镜→成片，每个环节都支持批量化操作。

啥意思呢？就是你不是一个镜头一个镜头磨的，而是整部剧的所有镜头一起排队生成。角色设定锁定之后，后续所有分镜自动引用同一套参考图，人物一致性的问题从流程层面就给卡住了。

适合谁用？有一定AI工具基础、想高效出多集连贯内容的人。如果你已经在用即梦或者ComfyUI做过单条视频，想升级到"量产短剧"，这个项目值得深挖。

不太适合纯小白——上手有门槛，需要理解分镜逻辑和提示词写法。

2. oowaoo AI影视Studio——网文直转视频

GitHub： github.com/saturndec/waoowaoo

这个项目瞄准的痛点特别精准：网文/小说怎么快速变成短剧？

目前AI短剧的爆款几乎都来自网文IP改编。但从一本几十万字的小说里拆出角色、场景、剧情线，再转化成可拍摄的分镜脚本，这个过程极其折磨人。我见过有人光拆剧本就花了一周。

oowaoo干的就是把这步自动化——你把小说文本扔进去，它自动解析人物关系、识别关键场景、提取剧情节点，然后生成带配音的完整短剧/漫画视频。

说白了，它帮你省掉的是"编剧+分镜师"这两个环节的活儿。

适合谁？手里有网文资源或者想做小说改编短剧的人。如果你在番茄小说、七猫上找到了一个好故事，又不想花大量时间拆解，这工具能让你两小时内看到初版成片。

注意一点：自动生成的剧本质量参差不齐，关键转折点和情绪高潮的处理往往需要人工微调。工具能帮你完成80%的工作，但最后那20%决定成败。

3. huobao-drama——小白的第一部AI短剧

GitHub： github.com/chatfire-AI/huobao-drama

如果前两个项目让你觉得有点复杂，那这个就是为你准备的。

huobao-drama的设计哲学就三个字：全自动。

你给它的输入可以是一句话、一个主题、或者一个粗糙的剧本大纲。它在后台自动完成这一整串操作：

剧本改写 → 角色提取 → 分镜拆解 → AI生图 → 视频生成 → TTS配音 → FFmpeg合成 → 字幕叠加 → 导出成片

你没看错，从一句"拍一个末世求生的短剧"到拿到一个可以直接发布的视频文件，中间不需要你碰任何第三方工具。

这是目前我见过的自动化程度最高的开源AI短剧方案。没有之一。

适合谁？零基础、想快速体验AI短剧制作全流程的人。哪怕你连分镜是什么都不知道也没关系，先用这个跑通一遍，建立对整个流程的感知，再去研究更精细的工具。

当然，全自动也意味着你对画面细节的控制力有限。要是你对画质和人物一致性有较高要求，还是得回头看Moyin Creator或者自己搭ComfyUI工作流。

三个项目怎么选？

简单粗暴的建议：

想先试试水，看看AI短剧到底是个啥——huobao-drama。十分钟出成片，体验一把再说。

手里有小说IP、想快速出内容测试市场——oowaoo。网文直转是它的强项。

已经有AI视频制作经验、想量产多集短剧——Moyin Creator。工业化流程，产能拉满。

三个可以搭配着来。比如先用huobao-drama跑个概念验证，确认题材有市场之后，再用Moyin Creator重新制作精品版本。

配套工具清单

光有开源项目还不够，得配上趁手的兵器。这里列一下我觉得当前最好用的几个：

视频生成：即梦AI（Seedance 2.0）。字节家的，目前公认最强。支持图生视频、文生视频、多图参考，新版3.5Pro做到了音画同步生成。每天有免费额度，小白先用这个练手。

角色一致性：可灵AI / 可灵O1。快手出品，2.6版本之后在人物记忆和跨镜头一致性上有了质的飞跃。年化收入超3亿美元——卖铲子的总是最赚钱的。

多集生产：商汤Seko 2.0。支持100集以内的连续剧创作，20多万创作者在用。合作短剧登顶过抖音AI短剧榜。

本地方案：ComfyUI + LTX工作流。开源免费，四宫格分镜直出12秒视频。不排队不审核，就是显卡要求高一些。适合有技术底子的人。

剧本/分镜：豆包 / Claude / Qwen。让大模型帮你写剧本、拆分镜、生成提示词。这步千万别自己硬写，AI出初稿你改，效率翻10倍。

配音：鱼声 / GPT-SoVITS / CosyVoice。TTS配音工具，有的支持声音克隆，有的支持情感控制。鱼声上手最简单，GPT-SoVITS效果最好但部署麻烦。

剪辑合成：剪映。国民级剪辑工具，2026年内置了AI文字成片、数字人、声音克隆，和即梦打通了，做短剧后期流程最顺。

一套跑得通的完整工作流

工具和项目都到位了，串一下流程。

第一步：写剧本。 用豆包或Claude写短剧剧本，结构就是"钩子—发展—反转—收尾"，单集500到1200字。题材选市场验证过的——末世、甜宠、悬疑、修仙都行，但如果你能找到差异化切口更好，修仙穿越已经卷成一锅粥了。

第二步：拆分镜表。 每段标注段号、画面内容、对白、景别、运镜、情绪。一段一个主动作，让AI帮你出初稿，自己微调。

第三步：定角色设定图。 即梦或NanoBananaPro生成角色图，锁定种子值。这步马虎不得，人物一致性是AI短剧最大的技术坎儿。

第四步：写提示词。 公式：主体+动作+场景+光影+镜头语言+风格+画质+约束。每段重复风格词画质词，角色段加"同一角色，服装一致，发型不变"。

第五步：Seedance 2.0逐段出视频。 9:16竖屏，上传角色参考图锁脸，每段出3到5条挑最好的。这步最费时间，也最看运气——有时候出来的画面惊艳得想截图发朋友圈，有时候人物直接长出六根手指。

第六步：剪映拼接。 按分镜顺序硬切为主，加配音、BGM、字幕。

第七步：发布。 红果短剧、抖音、快手，哪个有扶持就先投哪个。有人一个人做6集漫剧，靠平台分账赚了12万多。

几个过来人的坑

别忽视剧本。 工具再强也救不了烂故事。AI短剧的爆款几乎都是网文IP改编，不是因为技术牛，是因为故事本身经过了千万读者的市场验证。你要是原创，先去番茄小说看看什么题材在火。

角色一致性反复检查。 前一个镜头女主瓜子脸，下一个变国字脸，观众直接划走。每段都要引用同一张参考图，别偷懒。

镜头时长会失控。 剧本写了3秒，工具可能自动补成10秒15秒，成片时长翻倍。只能后期手剪。

别一上来就追求完美。 先用huobao-drama这类全自动工具跑通一遍，知道整个流程长什么样。再回头一个环节一个环节优化。我见过太多人在"选哪个AI模型生角色图"这一步纠结三天，最后一条视频都没出。

窗口期是有保质期的

重庆有个做短剧的人跟媒体说，他认识的近10家短剧公司里，还在做真人实拍的只剩两三家。有一家直接砍了导演岗位全转AI。

平台还在砸钱扶持，工具还在疯狂迭代，审核细则还没完全落地。这个时间窗口，给个人创作者留了一条缝。

但这条缝不会一直开着。等头部内容工厂把产能拉满、平台补贴退潮、监管规则收紧，留给单打独斗的人的空间就会快速收窄。
