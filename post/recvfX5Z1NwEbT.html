文章内容：
一、起因：Claude Code 源码泄露，我想做一份像样的研究报告
作者在 Claude Code 源码泄露后，决定制作一份深度技术报告，并面临将 Markdown 文件转换为出版级 PDF 的挑战。

二、"这能有多难？"——一个下午的崩溃之旅
作者尝试了多种方案，包括 Pandoc + LaTeX、WeasyPrint、md-to-pdf / Puppeteer 和 MiniMax 的方案，但都遇到了各种问题。

三、既然没有，那就自己做一个
作者最终选择了 reportlab 库来创建自己的 Skill，名为 lovstudio/md2pdf。

四、解决了哪些"别人没解决"的问题
md2pdf Skill 具有以下特点：
1. CJK 双层方案——字符级精确混排
2. 代码块保真——esc_code() 黑魔法
3. 全链路文档结构——从封面到封底
4. 智能表格列宽
5. 主题系统——10 种设计风格

五、对比：12 个竞品，逐一过招
作者对 12 个 MD→PDF 相关的竞品进行了对比，发现 md2pdf Skill 是唯一同时具备 Markdown 原生全功能排版 + CJK 专项支持 + 全链路文档结构 + 零重依赖的方案。

六、为什么不用 MiniMax 的 Skill？
作者认为 MiniMax 的 Office Skills 侧重通用文档平台，而 md2pdf Skill 侧重中文技术报告的全链路排版。

七、一个设计哲学的选择
作者认为一个好的 Skill 应该让 Agent 用起来像呼吸一样自然，不需要装浏览器，不需要学新格式，不需要配字体。

八、安装与使用
md2pdf Skill 可以通过 npx skills add lovstudio/md2pdf 一行命令安装，并在 Claude Code 里直接使用。

九、后记
作者认为 md2pdf Skill 的诞生是为了满足自己当时的需求，现在也可以被其他人使用。

文章总结：
本文介绍了如何使用 Claude Code Skill 生成专业的 PDF 报告，并介绍了 lovstudio/md2pdf Skill 的特点和优势。这个 Skill 能够接受标准 Markdown，对中文友好，并在 Claude Code sandbox 中直接运行，生成包含封面、目录、书签、水印、页眉页脚的专业 PDF。
