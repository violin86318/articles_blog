分享6个我觉得应该必装的Skills

掌握创造个性化Skills的能力是提升效率的关键。

核心摘要

核心观点： 通用AI能力正在贬值，而精准解决个性化需求的专属能力（Skills）成为关键竞争力。作者强推6个必装技能： 1. Frontend Design：根治AI前端设计“土味”，强制规划美学方向，优化排版/字体/动效。 2. 办公四件套：无缝读写/编辑Word、Excel、PPT、PDF文件，提升文档处理专业性。 3. Web Access：联网+操控浏览器（带登录状态），自动沉淀网站操作经验，越用越强。 4. PUA模式：专治AI“摆烂”，用7步检查清单强制切换思路，突破问题解决瓶颈。 5. Claude-mem：实现跨会话长期记忆，自动注入关键上下文。 6. Skill-Creator（核心）：允许用户将自身需求封装成专属Skill，实现个性化赋能。 结论： 最牛技能永远是“自己造的Skill”。Skills的价值在于将个人经验转化为可复用的智能工具，推动AI从通用助手进阶为专属生产力引擎。 注：全文未出现具体数据，核心聚焦个性化技能的价值升华。

正文整理

文章标题为《分享6个我觉得应该必装的Skills》。

文章内容主要介绍了作者认为对大家有用的六个Skills，并详细解释了每个Skill的功能和使用方法。以下是每个Skill的简要介绍：

  1. Frontend Design：这个Skill出自Anthropic官方，用于解决AI做前端页面时品味问题。它要求AI在写代码之前先想清楚一个大胆的美学方向，然后所有的排版、留白比例、字体选择、动效都要围绕这个方向来。装上这个Skill后，生成的页面排版会舒服很多。

  2. 办公必备四件套：包括docx、xlsx、pdf、pptx四个Skills，来自Anthropic官方。有了这个四件套，AI就能读取和编辑各种格式的文件，如Word、PPT、Excel、PDF等。装上这个四件套后，处理文档时会更加高效和专业。

  3. Web Access Skill：来自@一泽开发的联网Skill，将联网搜索和浏览器操控结合到一起。这个Skill可以通过Chrome DevTools Protocol直接连接本地的Chrome进程，带着登录状态，所以可以搜索和访问登录过的网站，如微博、小红书、B站等。它还能自动沉淀每个网站的操作经验，越用越顺。

  4. PUA：这个Skill专治AI摆烂问题。当AI在解决问题时遇到困难，可以开启PUA模式，它会强制打断AI的思路，让它执行一个7项检查清单，逼它换思路。PUA Skill能有效提高AI解决问题的效率。

  5. Claude-mem：这个Skill解决记忆持久化问题。它会自动记录每次对话里的关键信息，并在下次开新会话时自动注入相关的上下文。装上这个Skill后，Claude Code就能像小龙虾一样拥有长期记忆。

  6. Skill-Creator：这是作者认为最重要的Skill，它能帮助用户自己构建属于自己Skills的Skill。通过Skill Creator，用户可以将自己的需求和经验封装、沉淀成Skills，从而解决个性化问题。

文章最后总结了Skills的价值，认为通用技能在贬值，而知道自己需要什么的那个能力在升值。作者鼓励读者安装这些Skills，并强调最牛逼的Skill永远是下一个自己造的那个。
